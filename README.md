La API para Blog Personal es un servidor desarrollado con NestJS diseñado para potenciar un blog personal creado por un equipo de tres entusiastas estudiantes. Esta poderosa API proporciona un conjunto de endpoints robustos que permiten la gestión eficiente de contenido, comentarios y usuarios, brindando una base sólida para el funcionamiento de un blog dinámico y atractivo.

## Características Principales

- **Gestión de Artículos:** Crea, lee, actualiza y elimina artículos fácilmente a través de la API, proporcionando una interfaz intuitiva para administrar el contenido del blog.

- **Comentarios Interactivos:** Permite a los usuarios dejar comentarios en los artículos, fomentando la participación y la interactividad en la comunidad del blog.

- **Autenticación de Usuarios:** Incorpora un sistema de autenticación seguro para proteger áreas sensibles de la API, asegurando que solo usuarios autorizados tengan acceso a funciones privilegiadas.

- **Documentación Clara con Swagger:** Utiliza Swagger para documentar de manera clara y detallada todos los endpoints disponibles, facilitando la comprensión y el uso de la API por parte de los desarrolladores.

- **Escalabilidad y Mantenibilidad:** Desarrollada con NestJS, la API está estructurada de manera modular y sigue las mejores prácticas, lo que facilita la escalabilidad y el mantenimiento a medida que el blog crece.

## Uso

La API está diseñada para ser fácilmente integrada con el front-end del blog personal. Con endpoints intuitivos y bien documentados, los estudiantes pueden aprovechar al máximo las funcionalidades ofrecidas por la API en su proyecto de blog.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
